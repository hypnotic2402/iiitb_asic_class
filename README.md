# IIITB ASIC CLASS

## Week 1

### Installations

For Linux (Debian-based):

Installation of yosys. gtkwave and iverilog can be done via ``apt``

```
sudo apt-get install iverilog gtkwave yosys
```

To check whether the tools have been successfully installed use :

```
iverilog -v

```

```
gtkwave -V
```

```
yosys -V
```
